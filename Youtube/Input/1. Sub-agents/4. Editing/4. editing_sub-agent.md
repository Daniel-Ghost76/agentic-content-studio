# Editing Sub-Agent (Stage 4)

Claude executes this stage.

The editing stage has four sub-agents that run in sequence. Each hands off to the next. Read the relevant sub-agent file before starting any step.

## Pipeline

```
Raw footage (originals/)
    ↓
4a. Prep        → normalizes footage, syncs sources, enhances audio
    ↓
4b. Cut Edit    → transcribes, proposes cuts, executes after Daniel approves
    ↓
4c. Overlay ID  → maps all visual events (captions, zooms, overlays, Higgsfield) — Daniel approves map
    ↓
4d. Grade+Zoom  → applies color grade + bakes zoom keyframes in one render pass
    ↓
Stage 5 (Visuals) receives: {project_id}_cut_final.mp4 + {project_id}_overlay_map.md
```

## Sub-Agents

| Stage | File | Executor | Job |
|-------|------|----------|-----|
| 4a | `4a. prep_sub-agent.md` | Claude | Ingest, normalize, sync, audio enhance |
| 4b | `4b. cut_edit_sub-agent.md` | Claude | Transcribe, cut strategy, execute cuts |
| 4c | `4c. overlay_identifier_sub-agent.md` | Claude | Map all visual events — captions, zooms, HyperFrames, Higgsfield |
| 4d | `4d. grade_zoom_sub-agent.md` | Claude | Color grade + zoom keyframes in one render pass |
