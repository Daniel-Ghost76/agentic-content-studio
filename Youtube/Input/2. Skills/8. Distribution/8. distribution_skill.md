# Distribution Skill (Stage 8)

Claude executes this stage by default.

## What This Skill Does

Use this skill after Stage 7 publishing to prepare platform-specific distribution assets and summaries.

## Workflow

1. Read the YouTube URL and publishing metadata from the Stage 7 handoff.
2. Create concise distribution copy for selected platforms.
3. Save platform assets or notes to `Youtube/Output/8. Distribution/{project_id}/` using the canonical prefix, for example `{project_id}_linkedin.txt`.
4. Hand off any performance follow-up to Stage 9 Analytics.
