## Step 2 — Inventory `originals/`

```bash
ls -lh "Youtube/Output/4. Editing/{project_id}/originals/"
```

For each file, run:
```bash
ffprobe -v quiet -print_format json -show_streams "path/to/file"
```

Classify each source:
- **Talking head** — has voice audio, face visible, typically `.mov` from camera
- **Screen recording** — high-resolution display capture (2K+), may or may not have audio
- **Unknown** — flag to Daniel before proceeding

Write `{project_id}_manifest.json`:
```json
{
  "project_id": "{project_id}",
  "sources": [
    {
      "filename": "original_name.mov",
      "type": "talking_head",
      "duration_s": 1823.4,
      "resolution": "1920x1080",
      "fps": 30,
      "has_audio": true,
      "normalized_path": "normalized/{project_id}_talking_head.mp4"
    }
  ]
}
```

## Step 3 — Normalize Each Source

For each source, convert to: **1920×1080, H.264, AAC 48kHz, 30fps, SDR**.

Talking head normalization:
```bash
ffmpeg -i "originals/source.mov" \
  -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2,zscale=transfer=linear,tonemap=hable,zscale=transfer=bt709,format=yuv420p" \
  -c:v libx264 -preset slow -crf 18 \
  -c:a aac -ar 48000 -ac 2 \
  -r 30 \
  "normalized/{project_id}_talking_head.mp4"
```

Screen recording normalization (if present):
```bash
ffmpeg -i "originals/screen_rec.MP4" \
  -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2,format=yuv420p" \
  -c:v libx264 -preset slow -crf 18 \
  -c:a aac -ar 48000 -ac 2 \
  -r 30 \
  "normalized/{project_id}_screen_rec.mp4"
```

Verify each output with `ffprobe` — confirm resolution, fps, codec, audio. Fail loudly if spec is wrong.

## Step 4 — Sync Multiple Sources

**Single source:** Write trivial `sync.json` and skip to Step 5.
```json
{"version": 1, "sync_method": "single_source", "sources": {"talking_head": {"path": "normalized/{project_id}_talking_head.mp4", "offset_s": 0.0}}}
```

**Multi-source with compatible audio (both have voice/room audio):**
Extract audio from both sources and cross-correlate:
```bash
ffmpeg -i "normalized/{project_id}_talking_head.mp4" -vn -ac 1 -ar 16000 /tmp/th_audio.wav
ffmpeg -i "normalized/{project_id}_screen_rec.mp4" -vn -ac 1 -ar 16000 /tmp/sr_audio.wav
python3 -c "
import numpy as np, scipy.io.wavfile as wav, json
sr1, a1 = wav.read('/tmp/th_audio.wav')
sr2, a2 = wav.read('/tmp/sr_audio.wav')
a1 = a1.astype(float); a2 = a2.astype(float)
n = min(len(a1), len(a2), sr1*120)  # use first 2 minutes max
corr = np.correlate(a1[:n], a2[:n], mode='full')
lag = (np.argmax(corr) - (n-1)) / sr1
print(json.dumps({'lag_s': round(lag, 3), 'confidence': float(np.max(corr)/np.std(corr))}))
"
```

- If `confidence > 50`: use computed lag. Write `sync.json` with `sync_method: audio_correlation`.
- If `confidence <= 50` or screen recording has no useful audio: **Stop. Ask Daniel for manual timestamp.**

**Sync fallback (silent/incompatible screen recording):**
> "Automatic sync failed (confidence too low or no compatible audio). What timestamp in the screen recording corresponds to the moment your face cam recording started?"

Write `sync.json` with `sync_method: manual` and the provided offset.

```json
{
  "version": 1,
  "sync_method": "audio_correlation",
  "sources": {
    "talking_head": {"path": "normalized/{project_id}_talking_head.mp4", "offset_s": 0.0},
    "screen_rec": {"path": "normalized/{project_id}_screen_rec.mp4", "offset_s": 47.3}
  },
  "confidence": 84.2,
  "notes": ""
}
```

## Step 5 — Audio Enhancement with DeepFilterNet

Extract talking head audio, enhance, mux back:
```bash
# Install if not present
pip install deepfilternet 2>/dev/null

# Extract audio
ffmpeg -i "normalized/{project_id}_talking_head.mp4" -vn -ar 48000 -ac 1 /tmp/raw_audio.wav

# Measure before loudness
ffmpeg -i /tmp/raw_audio.wav -af ebur128=peak=true -f null - 2>&1 | grep "I:" | tail -1

# Run DeepFilterNet
python3 -c "
from df.enhance import enhance, init_df, load_audio, save_audio
from df.io import load_audio, save_audio
model, df_state, _ = init_df()
audio, _ = load_audio('/tmp/raw_audio.wav', sr=df_state.sr())
enhanced = enhance(model, df_state, audio)
save_audio('/tmp/enhanced_audio.wav', enhanced, df_state.sr())
print('Done')
"

# Apply two-pass loudnorm
ffmpeg -i /tmp/enhanced_audio.wav -af loudnorm=I=-14:TP=-1:LRA=11:print_format=json -f null - 2>&1 > /tmp/loudnorm_pass1.json
# (parse measured_I, measured_TP, measured_LRA, measured_thresh from output)
ffmpeg -i /tmp/enhanced_audio.wav \
  -af "loudnorm=I=-14:TP=-1:LRA=11:measured_I=<I>:measured_TP=<TP>:measured_LRA=<LRA>:measured_thresh=<thresh>:linear=true" \
  /tmp/enhanced_normalized.wav

# Mux enhanced audio back into talking head video
ffmpeg -i "normalized/{project_id}_talking_head.mp4" -i /tmp/enhanced_normalized.wav \
  -c:v copy -map 0:v -map 1:a \
  "audio/{project_id}_audio_enhanced.mp4"
```

Write `audio/{project_id}_audio_analysis.json` with before/after LUFS measurements.
