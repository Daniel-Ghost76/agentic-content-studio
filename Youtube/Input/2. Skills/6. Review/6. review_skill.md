# Review Skill (Stage 6)

Claude executes this stage by default.

## What This Skill Does

Use this skill to inspect finalized video packages, confirm thumbnails and metadata, and prepare assets for Stage 7 publishing.

## Workflow

1. Locate the target video folder in `Youtube/Output/6. Review  /`.
2. Confirm the video, transcript, thumbnail, and metadata files that exist.
3. Run thumbnail workflow when needed using `Youtube/Input/4. Resources/6. review_resources/thumbnail_workflow.md`.
4. Check that publishing metadata uses `.yaml` and matches the video filename exactly.
5. Summarize blockers or confirm the package is ready for Stage 7.

## Definition Of Done

- Review folder contents are summarized.
- Missing video, thumbnail, transcript, or metadata files are called out.
- Ready-to-publish packages have an exact `.mp4` + `.yaml` filename pair.
