# Ideation Rules

Claude owns Stage 1 by default.

## What this document is
A brief that gives the scripting agent everything it needs to write without inventing structure, facts, or direction itself. The ideation agent decides the angle, the hook, the beats, the analogies, and the framework phrase. The scripting agent writes the words.

## What this document is not
Not a script. Not a rough draft. The ideation agent decides structure — it does not write prose.

---

## Rules

**Reference videos must be real.** Verify every URL before including it. A broken link wastes Daniel's time.

**Search priority channels first.** Check the priority channels list before broadening search. If a priority channel has the relevant video, use it over a more popular video from elsewhere.

**The One Thing is one sentence.** If it takes two sentences, the idea isn't sharp enough yet.

**Hook is one line, decided.** Not options. Not candidates. One line under 30 words. The scripting agent uses it or improves on it — but it must have a starting point.

**Target Viewer is one sentence.** A real person with a real question, not a demographic category.

**Payoff must work backwards.** State what the viewer can do or say after watching. If the payoff is vague, the video has no spine.

**Beat Structure: Point + Visual + Target words per beat.** Point = what the beat establishes (one sentence). Visual = what is specifically on screen. Target words = approximate word count for that beat. No more, no less per beat block.

**Analogy Bank: minimum 2.** One per major concept. Label each by the concept it explains.

**Thesis Line Location names a beat.** Do not write the thesis line — that is scripting's job. Name which beat it lands in and summarize the insight in one sentence.

**Framework Phrase is 3 words or fewer.** It is the wrap — the thing the viewer remembers. Decided here, not in scripting.

**Specificity Bank uses real data.** Real numbers, real tool names, real data points only. Do not invent or estimate. If you cannot verify a number, leave a placeholder and flag it.

**Constraints are scope limits.** State what NOT to explain. Keep the video tight.

**Personal Angle and Vulnerability Moment are Daniel's to fill.** Suggest a placeholder — a likely angle based on the topic — but mark it `[DANIEL TO FILL]`. Do not fabricate personal experience.

**One document per idea.** One idea at a time unless explicitly instructed otherwise.

**Correct output location.** All PDFs go inside a canonical project folder: `Youtube/Output/1. Ideation/{project_id}/{project_id}_idea.pdf`.

**Correct project ID.** The project ID is created from the ordered idea number and title and is permanent for every later stage. Normal format: `{NN}_{lowercase_slug}`. Episode 1 is the explicit override `01-transcript-youtube`.
