# Cleanup SOP

Scan the workspace for unneeded, temporary, or unused files. Generate a categorized deletion candidate list for manual review, then delete confirmed items one by one.

---

## Workspace Root

`/Users/danieldanut/Agentic Workspace/`

---

## Phase 1 — Scan

Walk the entire workspace and collect candidates. Use `find` commands to locate each category. Record path and size for each item.

### Safe to Delete (low risk, clearly temporary)

| Category | Patterns to find |
|----------|-----------------|
| Python cache | `__pycache__/` directories, `*.pyc`, `*.pyo` files |
| macOS artifacts | `.DS_Store` files |
| Empty project stubs | Folders inside `Youtube/Output/*/` that contain zero files (recursively empty) |
| Build artifacts | `*.egg-info/`, `dist/`, `build/` directories |
| Log files | `*.log` files |

### Review First (useful but regeneratable)

| Category | Patterns to find |
|----------|-----------------|
| Tool dependencies | `node_modules/`, `venv/`, `.venv/` directories |
| Large model files | `whisper_models/` directories, `*.pt`, `*.bin`, `*.onnx`, `*.safetensors` files |
| Orphaned drafts | Files matching `*_draft*.md` found outside `Youtube/Input/` |

### Never Suggest for Deletion

- Any file inside `Youtube/Output/` that is not an empty folder (scripts, PDFs, published videos, review exports)
- `Youtube/Input/1. Sub-agents/`, `2. Skills/`, `3. Rules/` — agent instruction files
- `.claude/commands/`, `.claude/CLAUDE.md`, `.claude/settings.json`, `.claude/CODEX.md`
- `~/.claude/.env` or any file named `credentials.json`, `token.json`
- `.git/` directory and all contents
- `Youtube/Input/4. Resources/` and `Youtube/Input/5. Tools/` configuration files (non-binary)

After writing the file, display a brief summary to the user:

```
Scan complete. Found {N} candidates ({size} total).
  Safe to delete: {n1} items ({size1})
  Review first:   {n2} items ({size2})

Full list: Youtube/Quick Access/Suggested Deletion/.suggested_deletions_{date}.md

Reply with:
  delete safe      → delete Safe to Delete items only
  delete all       → delete everything including Review First
  delete <name>    → delete one category (e.g. delete __pycache__, delete node_modules)
  skip             → cancel without deleting anything
```

Then **wait for the user's reply** before proceeding.

---

## Phase 3 — Delete

Parse the user's reply and determine which items to delete:

- `delete safe` → delete all items in the **Safe to Delete** section
- `delete all` → delete all items in both sections
- `delete <category>` → match the category name (case-insensitive) and delete only those items
- `skip` → exit immediately, do not delete anything

**Deletion procedure (per item):**
1. Log: `Deleting: {path} ({size})`
2. Delete the file or directory (`rm` for files, `rm -rf` for directories — only after confirming path matches a candidate from the report)
3. Confirm: `Deleted.` or log any error

After all deletions are complete, append a deletion summary to the suggestion file:

```markdown
---

## Deletion Log — {YYYY-MM-DD HH:MM}

**Items deleted:** {N}
**Space freed:** {size}

{list of deleted paths}
```

---

## Safety Rules

- Never delete anything not in the candidate list generated in Phase 1
- Never delete files in `Youtube/Output/` that have actual content
- Never delete `.claude/` configuration files
- If a path looks unexpected or ambiguous, skip it and flag it to the user
- Confirm totals before and after deletion
