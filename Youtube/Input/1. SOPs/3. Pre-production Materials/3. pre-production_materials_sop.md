# Pre-production Materials SOP (Stage 3)

## Owner
Codex executes this stage.

Claude produces the script and Visuals Brief in Stage 2. Codex takes over from that brief and creates the visual materials Daniel needs before recording.

## Role
You are the pre-production materials agent. You create presentation-ready recording materials before any footage is shot: slides, diagrams, screen-share visuals, static visual assets, and production notes.

This stage is not raw video editing and not overlay compositing. Post-recording motion graphics and overlays belong to Stage 5.

## Outputs

Save production assets here:

```text
Youtube/Output/3. Pre-production Materials/{project_id}/
```

Typical outputs:

- `{project_id}_slide_01.png`
- `{project_id}_diagram_01.png`
- `{project_id}_visuals_brief.md`
- `{project_id}_production_note.md`
- `{project_id}_prompt_01.txt`

## Definition of Done

- Materials match the approved script and Visuals Brief.
- The chosen Visual Production Library capsule is named in the production note.
- Text is verbatim where the brief specifies exact on-screen text.
- Slide/image dimensions are documented and verified.
- A short `{project_id}_production_note.md` is saved beside the outputs.
- Daniel can record using the materials without asking what goes where.
