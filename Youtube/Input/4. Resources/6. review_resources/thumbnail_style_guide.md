# Thumbnail Style Guide

Reference image:
`/Users/danieldanut/Downloads/www.youtube.com_@nateherk_videos.png`

Use the reference as pattern inspiration only. Do not copy individual thumbnails.

## Core Look

- YouTube-native, high contrast, readable at small sizes.
- One immediate idea per thumbnail.
- Face is the primary human attention anchor when a face photo is available.
- Backgrounds should be dark, punchy, and simple enough that the title reads instantly.
- Use strong blocks of black, white, orange, teal, and occasional bright accent colors.
- Prefer visible app/tool/logo cues when the script is about software, AI tools, workflows, or automation.

## Text Rules

- Use 2-5 words of thumbnail text.
- Use big, bold, uppercase text.
- Avoid sentences.
- Prefer curiosity and contrast:
  - `COPY THIS`
  - `OLD vs NEW`
  - `IT'S EASY`
  - `STOP GUESSING`
  - `BUILT IN 1 HOUR`
- Text must remain readable on a phone-size YouTube grid.

## Visual Patterns

- Face plus app/logo grid.
- Before/after split.
- Old vs new comparison.
- One big tool icon plus one clear claim.
- Workflow diagram with 2-4 large nodes.
- Red X / green check contrast.
- Big number plus outcome.

## Avoid

- Long descriptive text.
- Thin fonts.
- Busy diagrams where every detail matters.
- Generic AI glow with no specific idea.
- Copying Nate Herk's exact layouts, thumbnails, or channel branding.
