# 03c — Proof Screen

**Sub-library:** `03_full_screen_overlays`  
**Status:** draft

## What It Is

A full-frame documentary-style frame that presents a screenshot, result, or output as evidence. The screenshot is the hero — a glass label panel identifies what it is. Used when the footage cannot carry the proof visually.

## When To Use

- Showing a tool output, result, or dashboard stat as proof
- Citing an external source or screenshot
- Presenting a before/after result side by side
- Any moment where "I'll show you" needs a clean frame around it

## When Not To Use

- When Daniel is already screen-sharing live footage (redundant)
- For results that can be stated verbally without visual proof
- When the screenshot contains sensitive or real personal data

## What Good Looks Like

- Full 1920×1080, `--bg` or very dark background
- Screenshot displayed in a glass-bordered frame, centred or slightly above centre
- Small glass label panel below or above the screenshot with a short descriptor
- Subtle entrance: frame fades in + slight scale from 0.98
- Documentary feel — not cinematic, not flashy. Evidence, not decoration.
