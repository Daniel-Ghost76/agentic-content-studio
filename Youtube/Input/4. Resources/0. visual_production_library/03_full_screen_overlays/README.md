# 03 Full-screen Overlays

Use this sub-library for visual moments that temporarily take over the full 1920x1080 frame.

Starter question:

```text
What full-screen visual are we building first: chapter break, system explainer, workflow map, proof screen, or transition visual?
```

Full-screen overlays should be deliberate. They should clarify a big idea, create a section break, or replace footage when the footage cannot carry the point.
