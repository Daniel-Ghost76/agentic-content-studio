# 04a — AI Tech HUD

**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft

## What It Is

An abstract, AI-flavoured HUD or data-flow video clip generated by Higgsfield. Used as b-roll during AI/tech concept explanations, video intros, or behind-text overlay backgrounds.

## When To Use

- Daniel is explaining an abstract AI or automation concept with no concrete footage
- An intro or transition needs a cinematic tech visual to establish energy
- B-roll is needed for a voice-over segment about Claude, agents, or pipelines

## When Not To Use

- When Daniel is on camera — this replaces footage, not overlays it
- When a concrete tool demo or real UI is the better proof (use `03c_proof_screen`)
- Avoid back-to-back — use maximum once per video section

## What Good Looks Like

- Dark, near-black background
- Subtle light streaks, data-flow lines, or atmospheric glow — not a busy dashboard
- Clean, premium, slightly cinematic
- No text, logos, or readable UI elements generated in the clip
- Motion should feel purposeful, not random looping noise
