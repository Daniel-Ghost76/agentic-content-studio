**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft — seeded from eRS3CmvrOvA (Nate Herk)

## What It Is

A Higgsfield clip used at the end of a video for calls to action. Two variants: (A) a browser window showing a community or resource page with a bold text header above it — used for "join my free community / get the resource"; (B) a YouTube player frame showing the video thumbnail and channel info, with an animated like/subscribe prompt — used as the end-card "watch next" or "like the video" CTA.

## When To Use

- Final 30–60 seconds of the video
- Directing viewers to a Skool community, free resource, or next video
- The "if you got value, like this video" moment

## When Not To Use

- Mid-video — CTA composites are end-of-video only
- As the only CTA — pair with a verbal CTA from Daniel on face cam first
- Without a specific destination — always link to something specific

## Variant A — Community/Resource Browser
- Bold white uppercase text header (`"FREE SKOOL COMMUNITY"`, `~80px`) above a browser window
- Browser window shows the community/resource page (Skool, Gumroad, Notion, etc.)
- Near-black background, teal accent glow at the browser window edges
- Duration: 4–6 seconds

## Variant B — YouTube Player End Card
- YouTube player frame showing the video, with the channel name/subscriber count below
- An animated like button (cursor moving toward the like button, button filling)
- Dark background, YouTube red logo top-left, white UI
- Duration: 4–6 seconds (enough to read the video title + see the like animation)
