# 04c — Abstract Concept

**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft

## What It Is

A Higgsfield clip for abstract or metaphorical visual moments — when the idea is real but the visual representation needs to be evocative rather than literal. Used for concepts like "AI thinking", "automation", "scale", "connection".

## When To Use

- Daniel is describing a concept that has no literal visual equivalent
- The video needs atmospheric breathing room before a key claim
- A metaphorical visual underscores a thesis or emotional beat

## When Not To Use

- When a literal demo or diagram serves better
- When the concept is specific enough for a workflow map or proof screen
- As filler — every Higgsfield clip should have a narrative reason

## What Good Looks Like

- Evocative but not random — the visual should suggest the concept
- Slow, deliberate motion — atmosphere, not action
- Dark and restrained — no saturated colour explosions
- Final frame is clean and holdable (editor can freeze it)
