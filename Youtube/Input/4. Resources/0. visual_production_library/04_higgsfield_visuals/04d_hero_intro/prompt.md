# Higgsfield Capsule Prompt: Hero Intro

**Sub-library:** `04_higgsfield_visuals`  
**Capsule:** `04d_hero_intro`  
**Tool:** Higgsfield video generation API  
**Status:** draft — seeded from eRS3CmvrOvA

---

## Prompt Template

```
Video opening hero shot. Near-black background (#09090b). 
Background layer: 10 small UI screenshot panels arranged in a loose asymmetric grid, 
softly blurred (gaussian blur), low opacity — suggest a collage of research or tool usage. 
Panels have subtle slow drift / parallax motion.

Foreground center: a large dark glass card (~900px wide), prominent, sharp.
Inside the card: "{product_name}" in large bold white text, with "{product_icon_description}" 
icon on the left. Below the text: a thin horizontal progress bar or metric reading 
"{metric_from} → {metric_to}" in smaller text.
Card has a thin teal glow at the top edge: rgba(16,185,129,0.6).

Overall palette: near-black, cool-neutral. Single teal accent.
Cinematic, premium, slow motion. Final frame holds clean.
Aspect ratio: 16:9. Duration: {duration_s} seconds.
```

---

## Variables

- `{product_name}` — the product or tool being covered (e.g. `"Claude Code"`, `"Cursor"`, `"Notion AI"`)
- `{product_icon_description}` — description of the product icon for Higgsfield (e.g. `"an orange-red asterisk / starburst icon on a dark rounded square background"`)
- `{metric_from}` — start value of the experience metric (e.g. `"0hrs"`, `"0 tests"`)
- `{metric_to}` — end value (e.g. `"400hrs"`, `"100+ skills"`)
- `{duration_s}` — clip duration in seconds (default: `5`, range: `4–7`)

---

## Generation Settings

- Aspect ratio: `16:9`
- Resolution: `1920 × 1080`
- Motion: slow, parallax on background panels, card is mostly static but may have subtle scale-in from 0.97→1.0
- Final frame: card fully settled, panels still drifting gently — must be cuttable on any frame

---

## Anti-list

- No text generated by Higgsfield inside the background panels (they should be recognisable as screenshots but not readable)
- No human faces in the background panels
- No bright colours — single teal accent only
- No animated logos or loading spinners
- Do not loop — single direction motion only
