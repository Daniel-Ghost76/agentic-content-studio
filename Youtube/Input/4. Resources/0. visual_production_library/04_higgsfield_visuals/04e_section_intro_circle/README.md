# 04e — Section Intro Circle

**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft — seeded from eRS3CmvrOvA (Nate Herk)

## What It Is

A Higgsfield clip that announces a new skill, topic, or section. A large teal-filled circle with the section name in white sits centred inside a teal-glow rectangle frame on a near-black background. Simple, bold, clean — functions as a chapter break card generated via Higgsfield.

## When To Use

- Transitioning into a new skill, tool, or section of the video
- Any "here comes the next one" moment — replaces a talking-head cut with a visual chapter marker
- Used once per major section (6 times in a 6-skill video)

## When Not To Use

- As an intro/outro — that's `04d_hero_intro` and `04o_cta_composite`
- Back-to-back without face cam between them
- When the section name is longer than 3 words — simplify the label

## What Good Looks Like

- Background: near-black, possibly faint radial teal glow emanating from behind the circle
- Outer frame: rounded rectangle, full-frame-width-ish (~1400px wide), thin teal glow border (`1.5–2px, rgba(16,185,129,0.6)`)
- Centre circle: `~260px` diameter, solid teal fill (`#10B981` or close), white section name text inside (`~28–36px`, bold, uppercase or title case)
- Circle may have a subtle scale-in animation (0.85→1.0 over the clip duration)
- Frame border pulses or fades in with the circle
- Duration: 2.5–4 seconds
- Final frame: fully settled, circle and frame sharp and stable
