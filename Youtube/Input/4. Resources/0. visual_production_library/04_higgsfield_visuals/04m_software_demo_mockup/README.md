# 04m — Software Demo Mockup

**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft — seeded from eRS3CmvrOvA (Nate Herk)

## What It Is

A Higgsfield clip of a laptop or MacBook frame (with screen showing real or realistic software UI) on a near-black background. An annotation pill or badge overlays the top of the laptop screen identifying the skill or feature being demonstrated. Used as a "proof of what this looks like in practice" visual without requiring actual screen recording.

## When To Use

- Showing what a tool or skill looks like when active inside Claude Code or another app
- Any "here it is running" moment where a mockup is cleaner than a raw screen recording
- When you want to frame a real screen recording inside a premium laptop frame

## When Not To Use

- When a real screen recording is available and more informative — this is a mockup fallback
- For abstract concepts (use `04c_abstract_concept`)
- When the phone/mobile context is more appropriate (use `04g_device_scenario`)

## What Good Looks Like

- Laptop: MacBook-style, silver/space grey, screen open, angled slightly (~10° tilt)
- Screen content: a real-looking dark-mode app (Claude Code VSCode, terminal, or browser) showing relevant activity
- Badge/pill: overlays the top-center of the laptop screen — e.g. "✓ Superpowers" in white on dark teal, pill shape
- Background: near-black, faint teal glow behind and beneath the laptop
- The laptop rests on a surface or floats slightly above a subtle reflection
- Duration: 4–6 seconds, mostly static, very subtle idle animation (screen glow, cursor blink)
