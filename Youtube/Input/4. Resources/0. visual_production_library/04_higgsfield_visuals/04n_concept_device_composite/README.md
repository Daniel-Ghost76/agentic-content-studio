# 04n — Concept Device Composite

**Sub-library:** `04_higgsfield_visuals`  
**Status:** draft — seeded from eRS3CmvrOvA (Nate Herk)

## What It Is

A Higgsfield clip combining a large bold concept headline at the top with a tablet or iPad mockup below it showing relevant code, documentation, or tool content on screen. Used when Daniel wants to make a thesis statement ("AUTOMATE BUSINESS") land visually while simultaneously showing the tool that enables it.

## When To Use

- A thesis or payoff statement needs to be shown visually as text, not spoken alone
- "Here's the capability" moments where the text claim is backed by a screen visual
- Transitional slides between sections where the core message needs reinforcing

## When Not To Use

- For regular section titles (use `04e_section_intro_circle` instead)
- When the device content is the focus — the text must also be prominent
- For more than 2–3 words in the headline

## What Good Looks Like

- Top half: large bold white uppercase text (`"AUTOMATE BUSINESS"`, `~80–96px`, letter-spaced, slight shadow)
- The last word may appear in a slightly different weight or have teal highlight
- Bottom: an iPad or tablet frame (landscape), screen showing dark-mode code or documentation (VSCode, Claude Code, a skill library)
- Very dark background, hint of ambient glow under the tablet
- No teal frame border (unlike other diagram capsules) — more like a product hero aesthetic
- Duration: 4–6 seconds
