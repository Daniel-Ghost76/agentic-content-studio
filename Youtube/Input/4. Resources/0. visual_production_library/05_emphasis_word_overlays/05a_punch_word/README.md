# 05a — Punch Word

**Sub-library:** `05_emphasis_word_overlays`  
**Status:** draft

## What It Is

A single large word that slams onto the screen, holds briefly, and exits. High-impact. Used for hook payoffs, key terms being named for the first time, or a single idea that needs to land hard.

## When To Use

- Daniel delivers the hook payoff and the word needs visual reinforcement
- A concept is being named and the name itself is the punchline
- A single word carries the thesis of a section ("DONE", "AUTOMATED", "FREE")

## When Not To Use

- More than one word — use `05b_two_word_contrast` instead
- Soft or nuanced moments — this has impact energy, not reflection energy
- More than once or twice per video (loses impact fast)

## What Good Looks Like

- Single word, very large (100–140px)
- Full-frame position: centre or slightly above centre
- Slam entrance: scale from 1.08 + opacity 0 → 1 in ~120ms — fast, punchy
- Short hold (1–2s)
- Clean exit: opacity 0 in 150ms
- Transparent canvas (composited over footage or chapter break bg)
