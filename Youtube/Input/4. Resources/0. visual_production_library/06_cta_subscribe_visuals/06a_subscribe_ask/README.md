# 06a — Subscribe Ask

**Sub-library:** `06_cta_subscribe_visuals`  
**Status:** draft

## What It Is

A subscribe prompt that feels native to Daniel's channel, not corporate. Small and respectful — it earns its place in the video rather than demanding attention.

## When To Use

- Mid-video at a natural pause after a strong value moment
- End of video before the outro

## When Not To Use

- Before value has been delivered — earn it first
- Multiple times in the same video
- During a fast-paced explanation or demo

## What Good Looks Like

- Small glass pill or card, bottom-centre or bottom-right
- Channel name + "Subscribe" prompt — clean, not cluttered
- Optional: small bell or tick icon (minimal, no YT branding copy)
- Entrance: gentle slide up from bottom + fade
- Auto-dismisses after 3–4 seconds
- Should feel like a quiet ask, not an interruption
