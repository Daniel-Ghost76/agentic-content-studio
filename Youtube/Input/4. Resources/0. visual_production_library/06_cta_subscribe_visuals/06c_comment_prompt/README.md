# 06c — Comment Prompt

**Sub-library:** `06_cta_subscribe_visuals`  
**Status:** draft

## What It Is

A visual prompt that surfaces Daniel's question to the audience, making the comment CTA feel designed rather than spoken only. The question appears on screen while Daniel asks it out loud.

## When To Use

- Daniel asks the audience a direct question and wants the comment CTA to land
- Near the end of the video after the main value has been delivered

## When Not To Use

- If Daniel is not asking a question — don't manufacture one
- Mid-video where it breaks the content flow

## What Good Looks Like

- Glass card, centred or bottom-centre
- Speech bubble or quote visual (optional, subtle)
- Daniel's question as the main text (36–44px, one or two lines)
- "Comment below ↓" as a small label underneath
- Entrance: fade + gentle rise from below
- Holds 3–5 seconds
