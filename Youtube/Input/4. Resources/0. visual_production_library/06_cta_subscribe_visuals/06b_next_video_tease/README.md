# 06b — Next Video Tease

**Sub-library:** `06_cta_subscribe_visuals`  
**Status:** draft

## What It Is

A small card teasing the next video, appearing bottom-right in the last 20–30 seconds of the video. Drives continued viewing without the end-screen interrupting the flow.

## When To Use

- Final section of the video before the outro
- When there is a clear next video to send viewers to

## When Not To Use

- Mid-video — too early, distracts from the current content
- When there is no specific next video to reference

## What Good Looks Like

- Small glass card, bottom-right, max 340px wide
- "NEXT →" label in small caps, then video title below
- Optional thumbnail image (if available)
- Enters from the right edge, holds for 15–25 seconds
- Should feel like a friendly pointer, not a banner ad
