# 02b — Stat / Metric Card

**Sub-library:** `02_partial_overlays_hyperframes`  
**Status:** draft

## What It Is

A glass panel built around a large number or measurable result. The number is the hero — the label exists to give it context. Used whenever Daniel quotes a proof point, result, stat, or metric.

## When To Use

- Quoting a result: "saved 4 hours a week"
- Citing a number from research or personal data
- Showing a before/after metric comparison
- Making a proof claim visual

## When Not To Use

- When the information is not a number or metric (use `02a_callout_card`)
- When you need multiple stats side by side (build a new capsule for that)
- When the number needs lengthy explanation (use a full-screen overlay)

## What Good Looks Like

- Glass panel, 300–440px wide
- Large number (64–80px) with optional unit suffix
- Small label below or above (caps, secondary colour)
- Optional prefix ($, ~, +, ×)
- Number counts up from 0 or slams in with a scale entrance
- Short hold (1.5–3s), clean exit fade
- Position: bottom-left, bottom-right, or top-right corner
