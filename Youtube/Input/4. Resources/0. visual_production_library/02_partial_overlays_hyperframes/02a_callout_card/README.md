# 02a — Callout Card

**Sub-library:** `02_partial_overlays_hyperframes`  
**Status:** draft

## What It Is

A small glass panel that appears on top of footage to surface a single piece of information — a term, a tool name, a short fact, a label for what Daniel is pointing at. The most commonly used overlay in the pipeline.

## When To Use

- Daniel mentions a tool or concept and the name should appear on screen
- A key point needs a visual anchor without interrupting the talking
- A UI element or workflow step needs a label callout

## When Not To Use

- When the information is more than 8 words (use `02c_step_list` or `03a_chapter_break`)
- When a number or metric is the hero (use `02b_stat_metric_card`)
- When you need to cover the full frame (use sub-library 03)

## What Good Looks Like

- Glass panel, ~320–420px wide
- One label line (small caps, secondary text colour)
- One value line (large, primary text colour, readable in 2 seconds)
- Entrance: slide in from the nearest safe-zone edge + fade
- Sits bottom-left or bottom-right of frame, never blocks face cam centre
- Panel never wider than 480px
