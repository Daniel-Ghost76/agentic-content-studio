# 07 Thumbnails

Use this sub-library for thumbnail concept patterns, references, prompts, and renderer recipes.

Starter question:

```text
What thumbnail pattern are we building first: face plus tool, before/after, old vs new, big number, proof/result, or workflow diagram?
```

Thumbnail capsules must prioritize readability at phone grid size, one immediate idea, and strong curiosity without misleading the viewer.
