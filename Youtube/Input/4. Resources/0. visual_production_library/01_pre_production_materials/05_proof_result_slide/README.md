# Proof / Result Slide

**Status:** draft
**Sub-library:** `01_pre_production_materials`
**Primary use:** showing that something happened, worked, shipped, scheduled, generated, or produced an output
**Default tool:** screenshot-first deterministic composition, with optional image generation for supporting scene
**Output format:** PNG/PDF

## When To Use

Use when Daniel needs to land evidence: final result, scheduled video, dashboard, output file, generated artifact, analytics, or proof of completion.

## When Not To Use

Do not use for abstract concepts or slides where no proof exists.

## What Good Looks Like

- The result is visible.
- The viewer can understand the proof quickly.
- Any highlight is restrained and specific.
- Text is short and supports the evidence.
