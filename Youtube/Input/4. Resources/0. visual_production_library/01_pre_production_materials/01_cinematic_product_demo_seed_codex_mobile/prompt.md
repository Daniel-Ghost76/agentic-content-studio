# Prompt

**Status:** draft

This prompt is a starting point only. It should be tested and rewritten with Daniel.

```text
Create a premium pre-recording presentation slide for a YouTube product/tool demo.

Format: portrait 1024x1536 or 9:16-safe vertical slide.

The slide should show a realistic product workflow with a phone and computer/laptop interface, but keep the design clean enough for Daniel to record over. The product UI should be the hero. Use a dark neutral background, restrained blue/teal accents, and large readable headline text.

Composition:
- large headline near the top
- short supporting line under the headline
- central realistic product/device visual
- optional one-line takeaway near the bottom

Style:
premium, realistic, clean, modern software/product demo, restrained cinematic lighting, high readability, not cluttered.

Avoid:
busy fake UI, tiny unreadable details, too much glow, overly dramatic lighting, generic sci-fi dashboards, distorted text, random logos, fake brand marks, cluttered backgrounds.

Variables:
Headline: {headline}
Support line: {support_line}
Bottom takeaway: {takeaway}
Product/workflow: {workflow_description}
```
