# Cinematic Product Demo - Seeded From Codex Mobile

**Status:** draft
**Sub-library:** `01_pre_production_materials`
**Primary use:** pre-recording slides for product/tool demos where the product interface or device workflow is the hero
**Default tool:** image generation plus deterministic text/layout pass when needed
**Output format:** portrait PNG/PDF for screen-share recording

## When To Use

Use for videos where Daniel is explaining a real tool, app, phone/computer workflow, launch, or setup process.

## When Not To Use

Do not use for dense teaching slides, abstract concepts, or moments where the text needs to carry the full explanation.

## What Good Looks Like

- A real-feeling device/workspace scene.
- Product UI is the main signal.
- Big simple headline.
- Clean dark composition.
- Minimal text.
- Premium but not overproduced.
