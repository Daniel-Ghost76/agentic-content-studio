# Pre-production Material Directions

This sub-library starts with five draft directions. None are approved yet.

Use these as the first menu when Daniel wants to build pre-production materials:

1. `01_cinematic_product_demo_seed_codex_mobile` - realistic product/tool visuals seeded from Codex Mobile, but not the final future style.
2. `02_minimal_thesis_slide` - one big idea, clean and readable, almost no supporting imagery.
3. `03_plain_educational_slide` - clear teaching slides with text/diagrams carrying the explanation.
4. `04_workflow_diagram_slide` - process, loop, architecture, before/after, roadmap, or system-flow visuals.
5. `05_proof_result_slide` - proof/result screens, screenshots, scheduled states, dashboards, or outputs.

## Working Rule

When building this library with Daniel, pick one direction, collect or generate references, then iterate until `prompt.md` produces repeatable results. Only then mark the capsule approved in `notes.md`.
