# Prompt

**Status:** draft

```text
Create a clear pre-recording workflow diagram slide.

Format: portrait or 16:9 based on the brief.

Show the relationship between the main parts of the system using simple nodes, labels, and arrows. Make the flow understandable at a glance. Use large text and avoid tiny interface details.

Composition:
- headline at top
- simple workflow diagram in the center
- optional one-line takeaway at bottom

Style:
clean, premium, readable, restrained, educational, not too technical-looking.

Avoid:
complex dashboards, tiny labels, too many arrows, heavy glow, fake code screens unless specifically requested.

Variables:
Headline: {headline}
Flow: {flow_steps}
Takeaway: {takeaway}
```
