# Workflow Diagram Slide

**Status:** draft
**Sub-library:** `01_pre_production_materials`
**Primary use:** process diagrams, loops, architecture, before/after, roadmap, and system-flow visuals
**Default tool:** deterministic renderer or image generation plus deterministic text pass
**Output format:** PNG/PDF

## When To Use

Use when Daniel needs to explain how parts connect or how something moves from input to output.

## When Not To Use

Do not use for emotional thesis lines or final proof screens.

## What Good Looks Like

- The relationship is instantly understandable.
- Labels are readable.
- Arrows/lines are simple.
- The diagram supports narration rather than replacing it.
