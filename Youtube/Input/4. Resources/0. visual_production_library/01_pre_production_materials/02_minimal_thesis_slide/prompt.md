# Prompt

**Status:** draft

```text
Create a minimal pre-recording thesis slide.

Format: portrait 1024x1536 or 16:9 depending on the recording brief.

Use one strong statement as the visual anchor. Keep the design clean, high contrast, and readable from a recorded screen. Use no decorative imagery unless it adds meaning.

Composition:
- one large thesis line or two-line contrast
- optional thin divider
- optional small label or section marker

Style:
quiet, editorial, confident, clean, high readability, modern sans-serif.

Avoid:
busy backgrounds, fake UI, long paragraphs, tiny text, heavy glow, generic gradients.

Variables:
Thesis text: {thesis_text}
Optional label: {label}
Mood: {mood}
```
