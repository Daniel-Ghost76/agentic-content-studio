# Minimal Thesis Slide

**Status:** draft
**Sub-library:** `01_pre_production_materials`
**Primary use:** one big idea, thesis line, section break, or memorable statement
**Default tool:** deterministic renderer
**Output format:** PNG/PDF

## When To Use

Use when the viewer should remember one idea and Daniel will explain the nuance verbally.

## When Not To Use

Do not use when the slide needs to teach a process, show proof, or display multiple examples.

## What Good Looks Like

- One sentence or two short lines.
- Very high readability.
- Strong whitespace.
- No visual clutter.
- The slide feels intentional, not empty.
