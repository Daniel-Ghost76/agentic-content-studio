# Prompt

**Status:** draft

```text
Create a plain educational pre-recording slide.

Format: portrait or 16:9 based on the brief.

The slide should teach one concept clearly using a headline, 2-4 short supporting points, and a simple diagram or comparison if useful. Prioritize readability and Daniel's ability to explain from the slide.

Style:
clean educational, restrained, modern, high contrast, simple layout, no decoration unless it improves understanding.

Avoid:
cinematic scenes, generic AI glow, busy dashboards, long paragraphs, tiny labels, complex charts.

Variables:
Headline: {headline}
Supporting points: {points}
Diagram need: {diagram_description}
```
