# Plain Educational Slide

**Status:** draft
**Sub-library:** `01_pre_production_materials`
**Primary use:** teaching concepts clearly with text, simple diagrams, or structured comparison
**Default tool:** deterministic renderer
**Output format:** PNG/PDF

## When To Use

Use for explainers where clarity matters more than visual spectacle.

## When Not To Use

Do not use for cinematic product demos or thumbnail-style punch.

## What Good Looks Like

- Clear hierarchy.
- Large readable text.
- Few bullets.
- Simple shapes and diagrams.
- Calm palette.
