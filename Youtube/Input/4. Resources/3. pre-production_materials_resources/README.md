# Pre-production Materials Resources

This folder stores Stage 3 briefs and reusable references for pre-recording assets.

Codex reads this folder when creating slides, diagrams, screen-share visuals, and production notes before Daniel records.

Primary files:

- `visuals_brief_format.md` — template for Visuals Briefs produced by Stage 2 scripting
- `{project_id}_visuals_brief.md` — episode-specific handoff brief

Stage 3 outputs go to `Youtube/Output/3. Pre-production Materials/{project_id}/`, and every artifact starts with the same project ID.
